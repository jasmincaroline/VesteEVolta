# Veste e Volta API

This project is a REST API developed using **ASP.NET Core** as part of the **CodeRDiversity Program**.

The project focuses on building a structured and scalable backend application, applying best practices such as layered architecture, use of DTOs, dependency injection, and automated testing.


## Technologies Used

- **.NET 10.0** - Latest .NET framework
- **ASP.NET Core Web API 10.0** - RESTful API framework
- **Entity Framework Core 10.0.0** - ORM for database access
- **PostgreSQL** - Relational database

## Prerequisites

- .NET 10.0 SDK
- PostgreSQL (latest version)
